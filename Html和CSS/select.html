1.选择器分组

.d1,.d2

选择器分组：各选择器之间用逗号间隔。
.d1,.d2 { color:red }

2.通配符选择器

一个* 号，对应于所有的元素

* { background:#e8e8e8; }

3.属性选择器

选择出具有某个属性的元素

*[name] { background:#fff; }

选择多个属性的方式：

*[name][age] { background:#666; }

选择多个属性的方式：

*[name = 3] { background:red; }

但只精确匹配该属性，即如果该元素还有其他属性也不能被匹配

4. 后代选择器

满足前后代的两个元素就会被选择

div[houdai] * {background :green}

5. 选择子元素

只有满足父子关系的两个元素就会被选择

div[zi] > * {background :orange}

6. 选择兄弟元素

选择相邻同级的元素，只选一个

用 + 表示

div[zi = '2'] > h2 + p {background :yellow}
