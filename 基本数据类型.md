## 1.Boolean
别的类型转为Boolean情况
```
  Number： 0-> false , others -> true
  String: ''-> false, others -> true
  Object: true
  Null : false
  Undefined: false
```
## 2.Number: 3种转换
  + Number
  + parseInt
  + parseFloat
### 区别：
```
  . 1，对空字符串'' ,
  Number('') === 0
  parseInt('') === NaN
  parseFloat('') === NaN
  . 2，对带数字字符串 '123.4abc'
  Number('123.4abc') === NaN
  parseInt('123.4abc') === 123
  parseFloat('123.4abc') === 123.4
```
