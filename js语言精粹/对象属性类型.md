### 1. 属性类型概述：
+  是什么
```
  ECMA-262第五版在定义只有内部才用的特性(attribute)时，描述了属性(property)的各种特征。 
  比如有对象：
  obj = {a: 1}
  属性类型指的是属性a的特征。
```
+  为什么
```
  ECMA-262定义这些特性是为了实现JavaScript引擎用的，因此在JavaScript中不能直接访问他们。
  // TODO: 属性类型如何在引擎中使用的呢？？
```
+  怎么实现
```
  为了表现特性是内部值，该规范把他们放在了两对方括号中。
  属性分为两种：数据属性和访问器属性。
```
